# My Skills


## Installation

To install the necessary dependencies, run the following command:

```bash
mkdir  -p ~/.copilot
git clone git@github.com:nghialuffy/my-skills.git ~/.copilot
```

## Configurate

- Upgrade your vs-code to the latest version

```
yay -S visual-studio-code-bin
```

- Ctrl + , to open settings
- Search Agent Skills
- Turn on: `Chat: Use Agent Skills`
- Restart vs-code using `Developer: Reload Window` command
- Go to Github Copilot > chat "What skills area available?"
